🔐 Infrastructure Alive Vault Era

Vault Variant B

The point where secrets stopped being scattered environment assumptions and became explicit infrastructure with a runtime path, a Web UI, and a human operator workflow.

What changed

Instead of hiding keys in local assumptions, the stack gained a Vault overlay, generated runtime env snapshots, and a restart-based injection model. This made secrets operationally visible and far less fragile.

Why it matters

It is a maturity exhibit. It marks the shift from "the system works on the machine that remembers the keys" to "the system has a reproducible secret path an operator can actually manage."

Surviving Genes
  • Managed secret namespaces instead of ad hoc `.env` sprawl
  • Visible operator workflow through Web UI
  • Runtime rendering and service restart discipline
  • A clearer line between configuration and real secrets
