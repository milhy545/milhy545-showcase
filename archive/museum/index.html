Curator Statement

Most portfolios only show what survived. This one also shows the fossils. The point is not nostalgia. The point is to document which experiments failed, which ideas mutated, and which constraints forced the current design to become better.

Classification
  • Alive Still part of the current stack.
  • Evolved Original form died, core idea survived.
  • Extinct Retired, but still worth learning from.
